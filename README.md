# Automatic Water Harvesting System

This project automatically collects rainwater using sensors and Arduino.

## Components Used
- Arduino UNO
- Rain Sensor
- Water Level Sensor
- Relay Module
- Water Pump
- Buzzer

## Working
- Detects rainfall
- Activates pump to store water
- Automatically stops when tank is full

## Applications
- Smart agriculture
- Rainwater conservation
- IoT-based water management
